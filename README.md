# LumenWave AV — Landing Page

This is a simple GitHub Pages landing page for an audio-visual company.

## Deploy on GitHub Pages
1. Push `index.html` to the repository root on the `main` branch.
2. Go to **Settings → Pages**
3. **Source:** Deploy from a branch
4. **Branch:** `main` / **Folder:** `/ (root)`
5. Wait for the Pages URL to appear.

## Customize
- Update company name, city, email
- Replace portfolio links
- Update social links (Instagram/YouTube/TikTok/LinkedIn)
